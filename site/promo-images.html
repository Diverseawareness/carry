How to export at 1024×1024

These render at 512px in the browser — Apple requires 1024×1024 PNGs.

Option A — DevTools: Inspect the promo element → Cmd+Shift+P → "Capture node screenshot" (enable 2× device scaling first).

Option B — Puppeteer: Run a headless Chrome script that sets viewport to 1024×1024 and screenshots each .promo element at 2× scale.

Option C — Manual: Rebuild in Figma at 1024×1024 using the same design.

Export specs: 1024×1024 PNG · RGB · no transparency · no rounded corners (remove border-radius before export).
